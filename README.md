# DiffJar

一个简易的批量 jar 包反编译对比小工具
依赖：git

使用

```bash
python main.py -i
```

进行程序初始化，将前一个 jar 包集合放在 f1rst 文件夹下，第二部分放在 s2cond

使用

```bash
python main.py -r
```

进行 diff jar 包
